## What Gets Collected

When a valid email is submitted, the app sends:

- `email` (required for submission)
- `ens` (optional, normalized to lowercase)
- `alias`
- `source` (`alias` or `end_screen`)
- `weeks`
- `archetype`
- `ts` (ISO timestamp)

It also stores locally in browser `localStorage`:

- `internoun_email`
- `internoun_ens`

## Supabase Setup

### 1) Get API credentials

From your Supabase project:

- **Project URL**
- **anonymous API key** (safe for frontend; do NOT use service role key)

### 2) Create table

Run this in Supabase SQL Editor:

```sql
create table if not exists public.email_captures (
  id bigint generated by default as identity primary key,
  email text not null,
  ens text,
  alias text,
  source text,
  weeks integer,
  archetype text,
  ts timestamptz default now()
);
```

### 3) Enable RLS + insert policy

```sql
alter table public.email_captures enable row level security;

create policy "allow anon inserts to email_captures"
on public.email_captures
for insert
to anon
with check (true);
```

> Optional hardening: add rate limits with Edge Functions or a server proxy.

## Environment Variables

Copy `.env.example` to `.env` and fill values:

```bash
cp .env.example .env
```

Required keys:

- `SUPABASE_URL`
- `SUPABASE_ANON_KEY`
- `SUPABASE_TABLE`

## Build (Inject `.env` into HTML)

Install dependencies:

```bash
npm install
```

Build static output with env injection:

```bash
npm run build
```

This generates:

- `dist/index.html` (with real Supabase values injected)

Deploy `dist/index.html`, not the root `index.html` placeholder template.

## Security Notes

- Never expose `service_role` or secret keys in frontend code.
- `anonymous API key` is expected to be public.
- Protect your data with RLS policies and validation.

## Local Run

Build first, then serve `dist/`:

```bash
npm run build
python3 -m http.server 8080 --directory dist
```

Alternative (quick visual check only): open root `index.html` directly.
Note this file still contains placeholders intended for build-time replacement.
